1. git clone repo
2. copy the `git_commit_inflator.vbs` into `C:\Users\<username>\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup`
3. open Task Scheduler
4. Create new task
5. name the task something like `Git_inflator` i dunno
6. Under actions click new, make sure to start a program, browse for the .vbs file you copied just now
